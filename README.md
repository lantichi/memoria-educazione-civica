# memoria-educazione-civica
Il Sito della Memoria è un luogo dove il passato non resta fermo: parla, interroga, chiede responsabilità. Tra tracce, nomi e storie, studenti e cittadini scoprono che la democrazia vive solo se qualcuno la custodisce. La visita diventa così un gesto di cura verso il presente e il futuro.
